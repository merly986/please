Просмотр истории изменений по сущности


функция, имитирующая обращение в бэк, пока мок - json\
возвращает общий список историю изменений
* изменения значений атрибутов (табл entity_attr_log)
* создание и удаление связи между сущностями (entity_entity, полу ts_created, ts_deleted)
* прохождение этапов (entity_stage)


компонент, который получает параметром ИД сущности (entity_id)
* отображает человекочитаемый список происходившего по сущности
* фильтрация и поиск на клиенте
